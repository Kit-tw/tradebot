# Trade Bot Website
* Built an LSTM model to predict asset prices using yFinance data.
* Integrated Socket.IO for real-time updates to enable live AI-driven trade decisions on MT4.
* Developed an e-commerce site for selling the bot, with secure payment and user authentication.
